# xLib  ![GitHub](https://img.shields.io/github/license/FunTc/xLib) ![Bintray](https://img.shields.io/bintray/v/funtc/Android/xLib?color=orange)
A basic library integrate and package some open source projects to facilitate project development. includes "Page Base UI", "EventBus", "OkHttp", "Retrofit", "Http Log"...



### **How to use?**

```groovy
implementation 'com.funtc.xlib:xLib:1.0.4'
```
